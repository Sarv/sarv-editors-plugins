## 项目介绍

这个仓库的内容是一个Sarv Office插件，也是OSPP-2024项目“编写插件，将证件照转换器与 Sarv Office 进行集成”的项目成果。

这个插件允许用户在Sarv Office内将个人照片转换为证件照。
